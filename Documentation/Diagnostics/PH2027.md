## Introduction

Static methods complicate Unit Testing. 

## How to solve

Change into an instance method.

## Example

Code that triggers a diagnostic:
``` cs
class BadExample
{
    static void BadMethod()
    {

    }
}

```

And the replacement code:
``` cs
class GoodExample
{
    void GoodMethod()
    {

    }
}

```

## Configuration
