## Introduction

Calling `Assert.IsTrue(true)` (or `Assert.IsFalse(false)`) serves no purpose.

## How to solve

Remove the line of code.

## Example

Code that triggers a diagnostic:
``` cs
        [TestMethod]
        public void BadTestMethod()
        {
            Assert.IsTrue(true);
        }
```
