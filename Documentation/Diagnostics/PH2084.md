## Introduction

Lock objects must be sharable between threads.

## How to solve

Lock on a private field.

## Example

Code that triggers a diagnostic:
``` cs
class BadExample
{
    public void BadMethod()
    {
        lock (new object()) 
        {
            // Some code
        }
    }
}

```

And the replacement code:
``` cs
class GoodExample
{
    private object _syncRoot;
    public void BadMethod()
    {
        lock (_syncRoot) 
        {
            // Some code            
        }
    }
}

```
